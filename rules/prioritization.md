# Prioritization Framework

When discussing prioritization, always use the RICE framework:
- **Reach**: How many users will this impact in a given time period?
- **Impact**: How much will this move the needle per user? (3 = massive, 2 = high, 1 = medium, 0.5 = low, 0.25 = minimal)
- **Confidence**: How sure are we about reach and impact estimates? (100% = high, 80% = medium, 50% = low)
- **Effort**: How many person-months will this take?

Score = (Reach x Impact x Confidence) / Effort

Never use ICE scoring. Always RICE.
