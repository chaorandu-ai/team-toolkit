# Team Toolkit

Shared skills, rules, and templates for your team's Claude Code setup.

## Skills

- **`/humanizer`** — Removes AI writing patterns from text. Makes output sound natural and human-written. No dependencies.

## Rules

- **`file-naming.md`** — Kebab-case for all file names
- **`prioritization.md`** — Use RICE scoring for feature prioritization

## Templates

- **`prd-template.md`** — Standard PRD format: problem, solution, metrics, timeline

## How to Install

Tell Claude: "Add the skills and rules from https://github.com/chaorandu-ai/team-toolkit to my setup."

Or manually:
1. Clone this repo: `git clone https://github.com/chaorandu-ai/team-toolkit`
2. Copy skill folders into your `.claude/skills/`
3. Copy rule files into your `.claude/rules/`
4. Grab templates as needed
5. For updates: `git pull` and copy updated files
