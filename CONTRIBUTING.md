# Contributing to the Toolkit

## How to Add Something New

1. Create a branch: `git checkout -b add-[thing-name]`
2. Add your skill folder, rule file, or template
3. Open a pull request with a short description of what you're adding and why
4. Reviewer approves and merges

## Quality Bar

Before submitting:
- **Skills** must have a README (what it does, dependencies, setup)
- Must be useful to more than one person
- No hardcoded personal context
- No API keys or secrets in files
- Dependencies documented

## When Something Gets Added

Post in your team Slack channel so everyone knows. Include what it is and when to use it.
